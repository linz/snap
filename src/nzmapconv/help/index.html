NZMapConv overview

NZMapConv is a program for converting coordinates between New Zealand coordinate systems used in topographic mapping. It converts between the following coordinate systems

  • NZMS260 map references defining positions in terms of the NZMS260 1:50,000 map series.
  • NZMG easting/northing defining positions in terms of the New Zealand Map Grid projection.
  • NZGD1949 latitude/longitude defining positions in terms of the NZGD1949 datum.
  • NZTopo50 map references defining positions in terms of the new NZTopo50 1:50,000 map series.
  • NZTM easting/northing defining positions in terms of the New Zealand Transverse Mercator 2000 projection
  • NZGD2000 latitude/longitude defining positions in terms of the NZGD2000 datum. Note that this is practically equivalent to WGS84 coordinates which are often generated by GPS receivers.

The program allows the user to enter coordinates for any of these input systems, and displays the equivalent coordinates in each of the other systems. The coordinates are remembered so that previous conversions can be easily reviewed. Coordinate conversions so that they can be easily reviewed. The program has several options allowing the user to modify the appearance of the program to suit their needs.

Note that the program has no facility for converting coordinates in a data file. To do that you can download the concord program from www.linz.govt.nz/downloadsoftware or use the online coordinate converter at www.linz.govt.nz/coordinateconversion. The concord program does not handle map references (NZMS260, NZTopo50).

For more information about New Zealand coordinate systems see www.linz.govt.nz/projectionsanddatums, and for more information about topographic maps see www.linz.govt.nz/topography/topo-maps.

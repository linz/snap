SNAP programs

The SNAP package comprises a number of programs for various managing various aspects of survey adjustment. The can be run individually from a DOS command line, or accessed using the snap manager program.

The programs can be categorised as follows:

Data preparation
dat2site Calculates approximate trial coordinates for stations based upon observations to them.
snapgeoid Adds geoid height information to a coordinate file using a gridded geoid model.
snapconv Converts a coordinate file between different coordinate systems.
snapmerge Merges two coordinate files
site2gps Generates a dummy GPS data file from a coordinate file for network accuracy analysis of a proposed survey.

Adjustment and analysis
snap Calculates new coordinates for stations using a least squares adjustment of observations.
snapplot Graphical tool for viewing and analysing adjustment results.

Reporting
snapspec Assigns coordinate orders to calculated coordinates using specifications in terms of absolute and relative accuracies.
snaplist Generates text reports of adjustment results.

Other utilities
concord Converts coordinates between different coordinate systems

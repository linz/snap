The stations list

The station list displays a list of the stations in the adjustment.

The list can be sorted by left clicking on the column heading by which it is to be sorted.

Left clicking on a row will select the corrresponding station on the map. If the Shift key is held down while left clicking a row, then the map will zoom in to the station. Double clicking will display detailed information about it in the details window.

The details window

The details window is used to display detailed information about specific stations, lines, and observations. The information displayed may include links (underlined) which can be clicked (left mouse button) to display the corresponding information. For example the window shown gives information about station 107. This includes a list of stations observed to from 107. Clicking on the connection to 96 will give information about the vector from 107 to 96, and a set of links to the specific observations of that vector.

If the Shift key is held down when clicking a link, then the map window will zoom in to the chosen station or vector.

Stations or vectors can be selected from the map window, stations can be selected from the stations list, and observations can be selected from the observations list.

The error information is modified by the error options dialog.

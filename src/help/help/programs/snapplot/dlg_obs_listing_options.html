The observation listing options dialog

The observation listing options dialog is used to select the columns and observations to display in the observations list. It is accessed from the Observations | Listing options menu item, or by right clicking the column headings in the observations list.

The items in this dialog are:

Fields to list Select the columns to include in the observation listing. This includes a standard set of columns and any observation classifications.
Observations to show Select which observations to include in the list. Options are to include all observations, used observations (those included in the adjustment), unused obserations (those not included in the adjustment), or rejected observations (those explicitely rejected from the adjustment).

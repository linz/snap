snapplot commands in the command file

The snap command file may include the following instructions for snapplot. These commands are ignored by snap.

plot configuration config_file_name

Specifies a snapplot configuration file to load when snapplot starts up. Snapplot will look for the file in the command file directory and snapplot program directory.

plot projection projection_crdsys_code

Specifies the projection coordinate system to use for the snapplot map.

plot background background_file

Specifies a file of background graphical features to plot on the map underlying the stations and observations.

plot offset_station station_code east_offset north_offset

Specifies an offset to be applied to the station coordinates when plotting the station on the map, generally used so that stations close together can be distinguished. The offset is always specified in metres east and north, regardless of the coordinate system of the adjustment.

The hide stations dialog

This dialog is used to choose which stations to hide and which to show on the map. It is accessed from the Stations | Hide stations menu item.

When a station is hidden then the station symbol, code, and name will not be displayed. The observations to and from hidden stations may also be hidden.

Note that the key can also be used to select some categories of stations to hide or show. Stations will only be shown if they are selected in both the key and in this dialog.

This dialog displays a list of stations from which hidden stations can be selected. It also allows categories of stations to be shown or hidden.

The items in this dialog are:

Station list This displays a list of stations in the adjustment which may be shown or hidden. The stations that are shown are highlighted in this list (have a coloured background) will be hidden on the map. Stations can be hidden or shown by clicking them in this list. Click on the column headers to sort the list according to name or code.
Stations to change This is used to hide or show a set of stations based upon some criteria. Stations matching the selected criteria can be added or removed from the list of hidden stations using the "Add" and "Remove" buttons. Other stations are unaffected - they remain hidden or shown as they were.

Many of the criteria (such as magnitude of the horizontal error, status, and so on) compare the station attributes with a test value. For most criteria the test value is entered into the threshold box.

For the "By status" criteria one or more status options can be chosen. Note that some options are exclusive. For example no station can be both fixed and adjusted horizontally, so if both these options are selected, then no stations will be affected.

Note: most of these options will not apply if a SNAP binary file has not been loaded by snapplot when it started.
Show observations to hidden stations If this option is selected then observations to and from hidden stations will be displayed. Otherwise they will not.
Add Stations matching the "stations to change" criteria will be added to the list of hidden stations
Remove Stations matching the "stations to change" criteria will be removed to the list of hidden stations
Clear All stations will be removed from the list of hidden stations
Done Exit the dialog box. The map will be updated to hide the selected stations

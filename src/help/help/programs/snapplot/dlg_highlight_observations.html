The highlight observations dialog

The highlight observations dialog is used to highlight a set of observations on the map according to some criteria. Highlighted observations are displayed in the highlight colour, which can be changed on the key This dialog is accessed from the Observations | Highlight observations menu item.

The items in this dialog are:

Don't highlight observations Turn off all highlighting of observations
Standardised residuals greater than threshold Highlight observations for which the a priori standardised residual is greater than the threshold value entered into the dialog.
A posteriori standardised residual greater than threshold Highlight observations for which the a priori standardised residual is greater than the threshold value entered into the dialog.
Redundancy factor less than threshold Highlight observations with a redundancy factor less than the threshold value entered into the dialog.
Rejected observations Highlight observations which have been explicitly rejected from the adjustment.
Unused observations Highlight observations which have not been used in the adjustment. This includes rejected observations and those which are not used because there was not enough information to calculate coordinates of the endpoint stations.
Between highlighted stations Highlight observations between highlighted stations. Stations are highlighted using the highlight stations dialog.
To or from highlighted stations Highlight observations which connnect to highlighted stations. Stations are highlighted using the highlight stations dialog.
Threshold value The value used as a criteria for highlighting based on standardised residuals or redundancy factor.

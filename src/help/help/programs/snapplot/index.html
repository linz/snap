snapplot - graphical view of adjustments

Snapplot is a graphical program for reviewing adjustment data and results. Before running an adjustment it can be used to provide a map view of stations and observations. After running an adjustment it can display statistical information about the generated coordinates and observations.

User interface

Once it has loaded a job snapplot presents the user with a map window showing the stations and observations in the network. The map has a key used to select which features to display and what colours to use.

Below the map is a textual window with tabs for displaying a list of stations, list of observations, or a details view which gives more information about specific stations or observations.

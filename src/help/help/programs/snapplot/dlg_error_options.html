The error options dialog

The error options dialog controls how observation errors, and station error ellipses and vertical error bars are displayed. This dialog is accessed from the Error | Error options menu item.

The items in this dialog are:

A priori errors All errors will be represented by a priori values .. that is based upon the purely specified errors of the observation, and not on how well they fit together and with the fixed coordinates.
A posteriori errors All errors will be represented by a posteriori errors .. that is scaled by the standard error of unit weight
Standard errors Error ellipses and height errors will be displayed a standard errors
Confidence limits Error ellipses and height errors will be displayed as confidence limits at the specified confidence level
Confidence level The confidence level to use when errors are displayed as confidence limits
Horizontal error exaggeration Specifies the exaggeration used to display error ellipses on the map. If autoscale is true then as the map scale changes the exaggeration will change so that the ellipses are drawn at an approximately consistent size. Note: These settings also apply to drawing the horizontal adjustment vector
Vertical error exaggeration Specifies the exaggeration used to display vertical error bars on If autoscale is true then as the map scale changes the exaggeration will change so that the ellipses are drawn at an approximately consistent size. Note these settings also apply to drawing the vertical adjustment.

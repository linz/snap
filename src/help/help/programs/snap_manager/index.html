snap_manager - environment for running adjustments

snap_manager is a configurable environment for running the snap programs. It is normally started from the start menu, or from a desktop icon.

It allows the user to select a SNAP command file for analysis, and run snap adjustments and other analysis programs. For tasks not available in the snap_manager menus use the Tools | DOS window option to open a command window. This will be set to the directory of the current command file.

The snap_manager environment is configured by the script file snap_manager.cfg stored in the program directory. The environment can customised for specific user requirements - for more information see snap_manager scripting.

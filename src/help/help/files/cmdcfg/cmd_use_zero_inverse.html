The use_zero_inverse command

use_zero_inverse [yes|no]

This command instructs SNAP to not calculate the inverse of the normal equations (which is the covariance of the parameters). This means that it cannot generate correct statistics for error ellipses and calculated errors of residual. Generally you would only use the for doing initial assessments of very large adjustments. You can achieve the same result with the snap -z option.

The ignore_observations command

ignore_observations selection_criteria

The ignore_observations command is used to identify observations that will be ignored. These observations are not read from the data file and take no part in the adjustment. Observations can be selected based upon observation type, observation date, stations used, and observation classifications.

The format of selection_criteria is described for the reject_observations command.

Observations can also be rejected or reweighted based on selection criteria. Rejected observations are not used to calculate coordinates but do still have residuals calculated, whereas ignored observations are not included in the adjustment at all.

Warning: If the criteria include selection based on data files, then the data_file commands must be placed earlier in the SNAP command file.

Examples of ignoring observations are:

ignore_observations @equipment=TAPE

ignore_observations date_unknown

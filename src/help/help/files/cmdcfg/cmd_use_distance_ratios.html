The use_distance_ratios command

use_distance_ratios option

This command specifies whether distance ratio data are to be processed as distance ratios, or treated as independent distance observations. The parameter of the command is:

option

can be "yes" or "no". If it is yes (the default) then sets of distance ratio data are processed as dependent observations with a common scale factor. If it is no then they are treated as independent slope distance observations.

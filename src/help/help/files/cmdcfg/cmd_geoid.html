The geoid command

geoid model [overwrite]

Defines a geoid model used to calculate geoid heights and deflection of the vertical for stations in SNAP. If the coordinate file already had geoid heights defined then they are not altered unless the "overwrite" option is specified.

The options for this command are:

model

Defines the name of the geoid model that will be used to calculate geoid heights and deflections of the vertical. SNAP will look for this file in the same directory as the command file, and in the SNAP program directory.

overwrite

If present specifies that SNAP will overwrite existing geoid information in the coordinate file.

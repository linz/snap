The output_coordinate_file command

output_coordinate_file coord_file_name

Specifies the name of the output coordinate file. By default this is based on the name of the primary coordinate file (the name in the first coordinate_file command), but with extension .new. However this can be changed with the output_coordinate_file command.

The output file name can be either an explicit file name, or a filename extension (eg ".newcrd"). If a file extension is given (if the name starts with ".") then the output coordiante filename is based on the command file but using this extension.

Warning if this command is used then the snap_manager program may not recognize the name of the new coordinate file.

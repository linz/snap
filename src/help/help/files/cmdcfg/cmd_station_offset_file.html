The station_offset_file command

station_offset_file stn_offset_file_name

Defines the name of the station offset file. The station offset file defines time series of offsets applying to the station coordinate that affect all observations at that station. For example it can represent a disturbance to a mark at a given date. It can also define the deformation time series that applieds at one or more time series. This can be used to define the deformation affecting stations that are outside the deformation model.

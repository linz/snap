The reject, accept, and ignore commands

reject [all] station_list
ignore [all] station_list
accept [all] station_list

Defines stations that are to be rejected, ignored, or accepted from the adjustment. Rejected station are not used in the adjustment, and observations to or from them are treated as rejected. Ignored stations are removed from the adjustment. Neither the stations nor any observations pertaining to them appear in the SNAP output. The accept command cancels reject or ignore commands. To select just few stations use ignore all and then accept the stations of interest.

all

if present specifies that all stations are affected

station_list

is a list of stations to be affected.

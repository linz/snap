The topocentre command

topocentre latitude longitude

Defines the topocentre of the network. This may be used to convert reference frame rotations and GPS errors between topocentric and geocentric reference frames. If no topocentre is defined then SNAP will calculate a midpoint based upon the extents of the coordinates in the data file. latitude and longitude should be entered in decimal degrees, for example

topocentre -41 170

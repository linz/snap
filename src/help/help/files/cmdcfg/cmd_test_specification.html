The test_specification command

test_specification name station_list

This command defines the list of stations for which a relative accuracy test will be applied. The command can be used several time, either to add new stations to the list applying for a specification, or to define sets of stations for different specifications. The relative accuracy will be tested between all stations associated with a specification, regardless of whether they are defined in the same or different test_specification commands.

SNAP also comes with a separate program snapspec that tests relative (and absolute) accuracy specifications. Unlike the test_specification command, snapspec is not used to test a specific set of stations. Instead it tries to find the largest possible group of stations that meet the specifications.

The parameters of the command are:

name

The name of the accuracy specification to apply (as defined in the specification command)

station_list

The list of stations for which the specification is to apply.

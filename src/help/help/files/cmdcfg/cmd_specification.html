The specification command

specification name confidence conf% horizontal habs mm hppm ppm vertical vabs mm vppm ppm

This command defines an accuracy specification that can be used to test the relative accuracy with which stations are positioned in an adjustment. There may be several specifications used in an adjustment (for example relating to different orders of station). Each specification is given a name which is used to refer to it in the test_specification command (which defines which stations it applies to).

The parameters of the command are:

name

The name used to refer to the specification.

conf

The percentage confidence to use for the test

habs

The absolute component of the horizontal tolerance in millimetres

hppm

The distance dependent component of the horizontal tolerance in part per million.

vabs

The absolute component of the vertical tolerance in millimetres.

vppm

The distance dependent component of the vertical tolerance in part per million.

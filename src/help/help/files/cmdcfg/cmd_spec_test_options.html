The spec_test_options command

spec_test_options options

This command defines options controlling how relative accuracy tests are applied and reported..

The options that may be specified are:

apriori

A priori errors will be used (the default)

aposteriori

A posteriori errors will be used

list_all

List all tested vectors individually

list_fail

List all vectors which fail the tolerance tests

list_none

Only print the summary information from the specifications test (the default).

The configuration command

configuration file_name

This command specifies the name of a configuration file to use for the adjustment. The configuration file is described in more detail above (see What is a configuration file, page 5-1). It is used for including groups of options which may apply to many adjustments. For example the configuration file could include a set of commands to get the preferred output format. The parameter of the command is:

file_name

specifies the name of a file from which additional configuration items will be read.

SNAP will look in the following directories in order to locate the configuration file:

the same directory as the command file

the user's directory. This directory is defined using either the SNAPDIR environment variable, or the -u option on the SNAP command line

the same directory as the SNAP program.

Point observation format

Each line contains an instrument station and the observations. This is used for observations such as latitude and longitude which involve just one station. The format is: 

inst_stn_definition   observation   observation
inst_stn_definition   observation   observation
inst_stn_definition   observation   observation
........

The instrument station definition inst_stn_definition defines the station code (the station code from the coordinate file) and the height of the equipment above the station. The preceding #data command may specify that the height is to be omitted (with the no_heights option), in which case the height is treated as being zero.

Each observation definition (observation in the formats above) includes the value of the observation (e.g. the length for a distance measurement). It may also include the error of the observation, and additional information such as refraction coefficient codes, observation classifications, and so on. There may be more than one observation type on each line. For example there may be both latitude and longitude observations.

The #gps_error_type command

#gps_error_type type

If you are explicitly defining the errors for each GPS observation rather than using default values then the gps_error_type defines the format they are entered in. type can be one of "full" (the default), "correlation", "diagonal", and "enu" (east, north, and up components).

The #projection command

#projection code

Defines the projection in which projection bearings are defined. code is the code of a coordinate system defined in the coordinate system definition file. SNAP will only accept the data if this code is the same as that in the coordinate file and the coordinate system defines a projection (that is, it is not a geocentric or latitude and longitude coordinate system).

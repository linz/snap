Bearing orientation errors

A bearing orientation error is a constant angular error applying to a group of azimuth or projection bearing observations. The error is measured in seconds of arc.

The main use for this error is in combining observations recorded from a number of survey plans. Each plan may have an orientation error resulting from the way the origin of the survey was established.

Each azimuth or projection bearing observation may have an associated orientation error code. Each code has an corresponds to an orientation error which may be either calculated or specified in the command file.

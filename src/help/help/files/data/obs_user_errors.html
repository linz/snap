User defined systematic errors

You can use SNAP to calculate parameters other than those specifically defined in the adjustment. For example you may believe that an EDM is introducing an unknown constant error into the data. You can specify this in a data file by defining the systematic error, say as EDM_CONSTANT, and then defining its influence on each observation. If you want to calculate EDM_CONSTANT in millimetres, then its influence on each distance observation is 0.001 metres.

Systematic errors are implemented using the "#systematic_error" and "#data" data definition commands in data files, and the "systematic_error" command in the command file.
